# Swagger Converter

This Node.js project converts `swagger.json` files generated by OpenAPI 2.0 to the latest OpenAPI 3.0 schema documentation.

## Features

- Converts OpenAPI 2.0 specifications to OpenAPI 3.0
- Maintains compatibility with existing API documentation
- Easy to use command-line interface

## Benefits

- Simplifies the migration to OpenAPI 3.0
- Ensures up-to-date API documentation
- Reduces manual conversion effort

## Installation

To install the dependencies, run:

```bash
npm install
```

## Setup

1. Clone the repository:

```bash
git clone https://github.com/yourusername/swaggerConv.git
cd swaggerConv
```

2. Install the dependencies:

```bash
npm install
```

## Usage

To convert a `swagger.json` file, use the following command:

```bash
node convert.js /path/to/swagger.json /path/to/output.json
```

Replace `/path/to/swagger.json` with the path to your OpenAPI 2.0 `swagger.json` file and `/path/to/output.json` with the desired output path for the OpenAPI 3.0 file.

## Example

```bash
node convert.js ./swagger2.json ./swagger3.json
```
